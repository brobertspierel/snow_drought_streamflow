Investigating streamflow conditions under different snow drought scenarios
==========================================================================
Guide:
^^^^^^
Documentation for a body of python scripts which are intended to collect USGS streamflow data and then ingest information from previous work on snow drought 
from SNOTEL, MODIS, VIIRS and Sentinel 1 and look at the relationship between datasets.

Dependencies: 
* climata <https://pypi.org/project/climata/>`_
* numpy 
* pandas 
* geopandas <https://geopandas.org/>`_
